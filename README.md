# 📖 Overview

FlowShield AI is an intelligent productivity assistant designed to help users complete important tasks before deadlines are missed.

Unlike traditional reminder applications, FlowShield AI analyzes a user's task, predicts potential productivity challenges, estimates completion probability, and generates an AI-powered execution strategy.

Instead of simply reminding users, FlowShield AI actively helps them plan, prioritize, and recover from distractions.

---

# 🎯 Problem Statement

### **Problem Statement 1**
## The Last-Minute Life Saver

Students, professionals, freelancers, and entrepreneurs frequently miss deadlines due to:

- Poor prioritization
- Context switching
- Overwhelming workloads
- Lack of execution planning
- Passive reminder systems
- Productivity burnout

Most productivity apps notify users **when it's already too late**.

FlowShield AI changes this by predicting productivity failure **before it happens**.

# 💡 Solution

FlowShield AI combines Google's Gemini AI with an interactive productivity dashboard to provide:

- AI-powered task analysis
- Productivity prediction
- Flow Score generation
- Deadline success estimation
- Smart task breakdown
- Personalized recovery strategy
- Motivational guidance

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/flowshield-ai.git
```

Move inside the project

```bash
cd flowshield-ai
```

Install dependencies

```bash
npm install
```

Create

```
.env.local
```

Add

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Start development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
